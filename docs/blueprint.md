# **App Name**: BookVerse API

## Core Features:

- Book Management API: RESTful endpoints for CRUD operations on books (title, author, genre, ISBN, price, description, stock, image URL).
- User Authentication API: Endpoints for user registration (customer/admin), login with JWT token generation, and secure password storage.
- Order Management API: Endpoints to create, retrieve, and update orders. Includes user details, book lists, order status, and payment status.
- Book Recommendations: Generative AI endpoint to provide personalized book recommendations based on user's order history and reading habits.
- Search and Filter Books: Enable efficient searching of books by title, author, or ISBN and filter books by genre. Implemented with backend querying.
- API Documentation: Auto-generated documentation using Swagger, detailing available endpoints, request parameters, and response formats.
- Error Handling: Provide informative error messages for request failures with appropriate HTTP status codes.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey a sense of intellect, sophistication, and the magic of stories. 
- Background color: Very light Indigo (#F0F8FF) for a soft and unobtrusive background.
- Accent color: Orchid (#DA70D6) to add a pop of color that's both modern and connected to the magical feeling.
- Body font: 'Literata', a transitional serif with a literary feel.
- Headline font: 'Belleza' - humanist sans-serif for personality, aligned to art and design
- Use minimalist icons representing book genres and user actions.
- Subtle animations for loading states and transitions between pages.
